## 1. タイトルの書き方（65文字制限）

### 1-1. 基本ルール

| ルール | 内容 |
|--------|------|
| 文字数上限 | **全角65文字**（半角は0.5文字換算） |
| 検索表示 | 一覧画面では**先頭約32文字**しか見えない |
| アルゴリズム | **先頭に置いたキーワードほど検索で重視される** |
| サジェスト連動 | ヤフオク検索窓のサジェストに表示されるワードをタイトルに入れるとヒット率UP |

### 1-2. タイトルの推奨構造（配置順）

```
[状態/訴求ワード] [ブランド名(日本語)] [ブランド名(英語)] [商品名/モデル名] [型番] [色/サイズ] [付加情報]
```

**配置の原則：**

| 位置 | 入れるもの | 理由 |
|------|-----------|------|
| **先頭（1〜10文字目）** | 訴求ワード + ブランド名 | 一覧で最も目に入る。検索アルゴリズムも先頭を重視 |
| **中盤（11〜40文字目）** | 商品名・型番・モデル名 | 具体的な検索キーワードでヒットさせる |
| **末尾（41〜65文字目）** | 色・サイズ・付加情報 | 補足。入りきらなければ説明文に回す |

### 1-3. 検索にヒットするためのキーワード配置テクニック

#### (A) ブランド名は日英両表記

ブランド名は「英語 + カタカナ」の両方を入れる。どちらで検索するユーザーも拾える。

```
良い例: LOUIS VUITTON ルイヴィトン モノグラム ネヴァーフルMM M40156
悪い例: ルイヴィトンのバッグです
```

文字数が足りない場合は、タイトルにどちらか一方、もう一方は説明文に記載する。

#### (B) 型番・品番は必ず入れる

型番を含む商品は、含まない商品と比較して**約35%高い価格で落札される傾向**がある。型番で検索するユーザーは「買う気」が高い。

```
良い例: ミズノ MIZUNO グローバルエリート 硬式 内野手用 1AJGH25413
悪い例: ミズノのグローブ 内野用
```

#### (C) 検索サジェストの活用法

1. ヤフオク検索窓にコアキーワードを入力
2. サジェスト（自動補完）に表示されるワードを確認
3. そのワードをタイトルに組み込む

例：「ミズノ グローブ」と入力 → サジェストに「硬式」「内野手用」「グローバルエリート」「ミズノプロ」が出現 → これらをタイトルに含める

#### (D) 「正規品」「本物」の明記

ブランド品では「正規品」「本物」の表記がある商品は、ない商品と比べて**約25%高い落札価格**になる傾向。

```
良い例: 正規品 GUCCI グッチ GGマーモント レザー ショルダーバッグ 443497 ブラック
```

#### (E) 状態を示すワードは冒頭に

「美品」「新品未使用」「レア」「限定品」などの訴求ワードはタイトルの**左側（冒頭寄り）**に配置。一覧でスクロールしたときに最初に目に入る。

```
良い例: 美品 HERMES エルメス ベアン スフレ 長財布 エプソン ブルーサフィール
悪い例: エルメス 長財布 ベアン スフレ 美品
```

#### パターン1：ブランド品（高額落札されやすい）

### 1-5. NG例（こうすると売れない）

#### NG1：キーワード不足

```
NG: 「バッグです。綺麗です」
```
→ ブランド名・型番・色・サイズ全てなし。検索に一切ヒットしない。

#### NG2：キーワードスパム（無関係なワード詰め込み）

```
NG: 「バッグ GUCCI CHANEL HERMES LOUIS VUITTON PRADA 高級 ブランド」
→ 実際はノーブランド品
```
→ 2025年以降、ヤフオクのアルゴリズムは不自然なキーワード詰め込みをペナルティ対象に。アカウントの信頼性低下・検索順位降格のリスク。

#### NG3：記号の乱用

```
NG: 「★☆★☆★ 超お買い得！！！ ★☆★☆★ バッグ」
```
→ 記号で文字数を浪費。重要なキーワードが入らない。アルゴリズムも低品質と判断。記号は**1〜2種類、合計2〜3個まで**に抑える。

#### NG4：文字数の大幅な余り

```
NG: 「ミズノ グローブ」（わずか8文字）
```
→ 65文字中57文字が未使用。型番・状態・色・サイズ等を入れるべき。

#### NG5：先頭に無駄な文言

```
NG: 「見てください！お得な商品です！ GUCCI バッグ」
```
→ 先頭14文字が検索に無関係。ブランド名・状態を先頭に。

#### NG6：英語とカタカナのどちらか片方だけ

```
NG: 「Louis Vuitton Neverfull MM」（カタカナなし）
NG: 「ルイヴィトン ネヴァーフル」（英語なし）
```
→ 片方だけでは検索の取りこぼしが発生。

### 1-6. TakeBackのタイトル先頭マーク対応

CLAUDE.mdによると、出品担当マークがタイトル先頭に付く：

| コード | マーク | 担当者 |
|--------|--------|--------|
| KH | 〇 | 林和人 |
| YY | ▽ | 横山優 |
| TK | ◇ | 鶴岡 |

**マーク + 半角スペース = 2文字消費** → 実質63文字でタイトルを作成する必要がある。

## 2. 商品説明文の書き方

### 2-1. 購入意欲を高める3部構成

売れる商品説明は **「キャッチコピー」「ボディコピー」「クロージングコピー」** の3パートで構成する。

#### (1) キャッチコピー（冒頭3行）

最初の3行で読み手の注意を引く。スマホでは最初の数行しか「もっと見る」を押さずに読めない。

```
◆ 正規品・鑑定済み ◆
GUCCI GGマーモント レザー ショルダーバッグ 443497
定価¥198,000 → 数回使用のみの極美品です
```

**ポイント：**
- 正規品・鑑定済み等の安心材料を最初に
- 定価との比較で「お得感」を演出
- 状態の良さを端的に

#### (2) ボディコピー（商品詳細）

**必ず入れるべき要素：**

| 要素 | 記載例 | 重要度 |
|------|--------|:------:|
| ブランド/メーカー | GUCCI（グッチ） | 必須 |
| 正式商品名 | GGマーモント スモール ショルダーバッグ | 必須 |
| 型番/品番 | 443497 | 必須 |
| カラー | ブラック | 必須 |
| サイズ（実寸） | 約W26×H15×D7cm ストラップ長さ55cm | 必須 |
| 素材 | レザー（牛革） | 必須 |
| 商品の状態 | 表面：微細なスレあり / 角：目立つ傷なし / 金具：光沢あり | 必須 |
| 付属品 | 保存袋、ギャランティカード、購入時レシート | 必須 |
| 購入時期/場所 | 2023年 正規店購入 | 推奨 |
| 購入価格（定価） | 定価 ¥198,000（税込） | 推奨 |
| 使用頻度 | 5回程度使用 | 推奨 |
| シリアルナンバー | 写真でご確認ください | ブランド品は推奨 |

**状態の書き方のコツ：**

悪い例：
```
中古品です。多少の使用感あり。
```

良い例：
```
【商品の状態】
・外装：角に微細なスレが2箇所ございますが、正面からは目立ちません
・内装：目立つ汚れなく、きれいな状態です
・金具：小傷なし。光沢が残っています
・ファスナー：スムーズに開閉します

※中古品のため、神経質な方はご遠慮ください
```

→ 「良い印象の情報を後に持ってくる」ことで、最後に良い印象が残る。

#### (3) クロージングコピー（購入を促す情報）

```
【発送について】
・佐川急便にて発送いたします
・送料は落札者様ご負担です（下記送料表参照）
・入金確認後、1〜2営業日以内に発送いたします

【送料表】
関東：800円 / 中部・関西：900円 / 東北・中国：1,000円
四国・九州：1,100円 / 北海道：1,300円 / 沖縄：1,800円

【ご注意】
・お使いのモニターにより実際の色味と異なる場合がございます
・中古品にご理解のある方のご入札をお願いいたします
・ノークレーム・ノーリターンでお願いいたします
・評価不要の場合はお取引時にお知らせください
```

### 2-2. 推奨文字数

- **最適文字数：600〜1,000文字**
- 短すぎると情報不足で入札されない
- 長すぎると読まれない（スマホユーザーが大半）
- 箇条書き・見出し・空行を活用して可読性を上げる

### 2-3. 中古品・ヴィンテージ品特有のコツ

| テクニック | 説明 |
|-----------|------|
| **購入時期・使用頻度を書く** | 落札者が「どのくらい使われたものか」判断できる |
| **傷・汚れは正直かつ具体的に** | 「右下角に1cm程度のスレ」のように場所と大きさを明記 |
| **傷の写真は必ず撮る** | 文章だけでは伝わらない。クレーム・返品防止にもなる |
| **付属品の有無を明記** | 箱・保存袋・ギャランティカード・タグ等。あれば高額になる |
| **経年変化をポジティブに表現** | 「味わい深いエイジング」「アンティーク特有の風合い」 |
| **時代・年代を特定** | 「昭和30年代」「1960年代」「大正期」など |
| **寸法は必ず実測** | ヴィンテージは規格外サイズが多いため必須 |

### 2-4. 現行品のコツ

| テクニック | 説明 |
|-----------|------|
| **定価を明記** | 「定価¥●●,000」と書くことでお得感を演出 |
| **型番で検索できるように** | 現行品は型番で検索されることが多い |
| **購入店舗・保証の有無** | 「ビックカメラ購入」「メーカー保証2027年3月まで」 |
| **動作確認済みを明記** | 家電は「全機能動作確認済み」が入札の安心材料 |
| **付属品を全てリスト化** | 説明書・リモコン・ケーブル・箱など、一つずつ列挙 |
| **Amazon/楽天の現在価格と比較** | 「Amazon現在価格¥●●,000 → 当オークション●●,000円スタート」 |

### 2-5. まとめ売りのコツ

| テクニック | 説明 |
|-----------|------|
| **「まとめて○点」を冒頭に** | 点数がわかると「お得感」が伝わる |
| **代表的な商品を数点ピックアップ** | 全部は無理でも目玉商品3〜5点は個別に説明 |
| **総定価を計算して明記** | 「定価合計 約¥●●●,000相当」 |
| **内容物リスト（表形式推奨）** | No./ブランド/商品名/状態/参考価格 |
| **「バラ売り不可」を明記** | 問い合わせ削減 |
| **ターゲットを明確に** | 「リサイクルショップ様」「同業者様歓迎」「部活の予備に」 |
| **重量・箱数を記載** | 送料の見積もりに必要 |

### 4-4. CSVテンプレートの作り方（手順）

1. オークタウンにログイン
2. 「出品する」→「個別商品登録」
3. 1商品分の情報を全て入力
4. 画面下部の「CSV形式でダウンロード」をクリック
5. ダウンロードしたCSVをExcelで開く
6. 1行目（ヘッダー）はそのまま残す
7. 2行目のデータを出品数分コピーして各商品の情報に書き換え
8. エンコーディング「日本語（Windows-932）」で保存
9. CSVファイルと画像ファイルを1フォルダにまとめてZIP圧縮
10. オークタウンの「一括出品」からZIPをアップロード

**注意：** TakeBackは将来的にヤフオクビジネスアカウント取得後、公式ストアCSV（BOM付きUTF-8）に移行予定。現在のBot（yahooauction_sheet.gs）はオークタウンVer.1.17準拠のCSVを出力する。
