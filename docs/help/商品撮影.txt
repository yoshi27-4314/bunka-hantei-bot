# 1　このチャンネルの役割


## 商品撮影チャンネルとは


分荷判定で確定した商品を、管理番号と紐づけてGoogle Driveに写真保存するチャンネルです。


  保存された写真は、出品チャンネルでのタイトル・説明文生成や、梱包・発送時の確認に使われます。

  写真の品質と管理番号の正確な紐づけが、後工程のスタッフ全員の作業精度に直結します。


## 撮影作業が事業全体に与える影響


✅ 良い写真・正確な紐づけ
    
- 出品タイトル・説明文をAIが自動生成できる
- 購入者が安心して入札・購入できる
- クレーム・返品リスクが下がる
- 梱包担当が商品を迷わず識別できる

  
  
    ⚠️ 写真が不足・番号が違う
    
- 出品担当が再撮影を依頼しなければならない
- 別の商品の写真で出品されるリスク
- クレームや返品につながる
- 全工程が止まる


## 全体の中での位置づけ


済み分荷判定 → ここ商品撮影 → 次出品保管 → 出品中 → 梱包出荷 → 出荷済み


## 担当AI：白洲次郎


このチャンネルのAIは 「白洲次郎」 として返答します。

  GHQとも一歩も引かなかった反骨の紳士。口調は短く的確です。

  「〜だ」「悪くない」「それでいい」など、簡潔な返答が特徴です。

# 2　撮影作業の流れ（4ステップ）


作業場所：撮影確認チャンネル（白洲次郎）

  テプラや管理番号を投稿してセッションを開始し、写真を送って「完了」で締めます。


- セッション開始 ─ テプラ写真 または 管理番号テキストを新規メッセージとして投稿する → AIが管理番号を認識してセッションを開始。使えるコマンドも表示されます
- 商品写真を投稿 ─ セッションのスレッドに商品写真を送る（複数枚OK） → Google Driveに自動保存。Botから確認チェックリストが届きます
- 写真を確認する ─ Botのチェックリストを見ながら投稿した写真を確認する → 問題なければ「完了」、追加があれば写真を追加投稿、撮り直す場合は「やり直し」
- 完了 ─ スレッドに 完了 と入力する → Monday.comのステータスが「撮影済み」に更新されます


⚠️ 重要：写真は必ず「スレッド内」に送ってください

  セッション開始のメッセージに対して「返信」する形で写真を送る必要があります。

  新しいメッセージとして送ると、別のセッションとして認識されません。

# 3　セッション開始の方法


## 方法A：テプラ写真を投稿する（推奨）


管理番号が印刷されたテプラのラベルを撮影し、チャンネルに投稿します。
  
- 商品に貼られたテプララベルを撮影する
- 撮影確認チャンネルに新規メッセージとして投稿する
- AIが自動でラベルの管理番号を読み取り、セッションが始まる

  テプラ画像はGoogle DriveのフォルダにもOCR前に自動保存されます（01_テプラ.jpg）。


## 方法B：管理番号をテキストで入力する


テプラがない場合や読み取りに失敗した場合は、管理番号を直接テキストで投稿します。
  
    
      ✅ 正しい形式
      
```
2603-0001
```

      管理番号のみを1行で入力
    
    
      ❌ 認識されない例
      
```
管理番号：2603-0001
```

      
```
洗濯機の撮影です
```

      余分なテキストがあると認識できない


## セッション開始後にBotから届くメッセージ

  📸 撮影セッション開始

  ━━━━━━━━━━━━━━━━

  ━━━━━━━━━━━━━━━━

  📌 作業手順


  　① このスレッドに商品写真を投稿

  　② Botの確認メッセージが届いたら写真をチェック

  　③ 問題なければ `完了` と送信


  ━━━━━━━━━━━━━━━━

    　② または管理番号をテキストで入力

    　　例） 2603-0001

# 4　写真の撮り方ガイド


## 基本の撮影ポイント


購入者が「実物を見た」と感じられる写真を撮ることを目標にします。

  きれいに見せようとするより、現状を正直に写すことが大切です。


| カット | 撮影内容 | ポイント |
| --- | --- | --- |
| ① 全体正面 | 商品の正面全体 | 明るい場所で真っ直ぐ。商品全体が入るように |
| ② 全体背面 | 商品の背面・裏側 | 型番シール・スペックシールも写すと◎ |
| ③ 側面・上面 | 左右どちらか・上面 | 立体感が伝わるように角度をつけて |
| ④ 傷・汚れ | 目立つ傷・汚れのクローズアップ | わかりやすく近づいて撮る。隠さないこと |
| ⑤ 付属品 | リモコン・ケーブル・取扱説明書など | 揃えて並べて一緒に撮る |
| ⑥ 型番・シリアル | 型番・製造年・シリアルナンバーのシール | 文字が読めるくらいクローズアップ |


💡 スマートフォンで十分です。最低4〜6枚を目安に撮影してください。枚数が多いほど出品の精度が上がります。


## 撮影時の注意点


✅ 良い写真の条件
    
- 明るい場所で撮影している
- 商品が画面の中央にある
- ピントが合っている
- 指や影が写っていない
- 傷・汚れを正直に写している

  
  
    ❌ 避けるべき写真
    
- 暗くてよく見えない
- ぼやけている
- 商品の一部が切れている
- 傷を隠すような角度で撮っている
- 背景が散らかっている


⚠️ 傷・汚れは必ず写してください

  状態ランクで「C（傷汚れあり）」と記録された商品は、必ず傷のアップ写真が必要です。

  写真と実物の状態が違うと、購入者からのクレーム・返品の原因になります。

# 5　コマンド一覧


⚠️ すべてのコマンドは「スレッド内」で入力してください。

  セッション開始のメッセージに返信する形で入力します。新しいメッセージとして送っても動作しません。


## 通常コマンド


| 入力する内容 | 動作 | 使うタイミング |
| --- | --- | --- |
| 商品写真を投稿 | Google Driveの管理番号フォルダに自動保存。Botから確認チェックリストが届く | スレッド内 |
| 完了 | 撮影完了。Monday.comのステータスが「撮影済み」に更新され、スプレッドシートに記録される | 写真確認後 |
| やり直し | Driveの商品写真を全て削除して1枚目から撮り直せる状態にする テプラ画像（01_テプラ.jpg）は削除されない | 写真が不満なとき |
| キャンセル / 中断 | 撮影作業を中断する（写真はDriveに残る） 再開するときは管理番号を新規投稿してセッションを開始する | 作業を中断したいとき |
| 削除 | Monday.comのステータスを「要確認」に戻す（2ステップ確認あり） | 登録自体をやり直すとき |

# 6　Google Driveへの自動保存について


## フォルダ構成


投稿された写真は、以下の構成でGoogle Driveに自動的に整理されます。
  
```
TKB｜A｜自社在庫（ヤフオク・eBay・shopify）
　└── 2603/（年月フォルダ：例 2026年3月）
　　　└── 2603-0001/（管理番号フォルダ）
　　　　　├── 01_テプラ.jpg　← テプラ写真
　　　　　├── 02_商品.jpg　　← 1枚目の商品写真
　　　　　├── 03_商品.jpg　　← 2枚目の商品写真
　　　　　└── 04_商品.jpg　　← 3枚目の商品写真 ...
```


## ファイル命名ルール


| ファイル名 | 内容 | 保存タイミング |
| --- | --- | --- |
| 01_テプラ.jpg | テプララベルの写真 | セッション開始時（テプラ投稿の場合） |
| 02_商品.jpg 03_商品.jpg... | 商品の写真 | スレッドに写真を投稿するたびに順番に採番・保存 |


テキストで管理番号を入力してセッションを開始した場合、01_テプラ.jpg は作成されません。02_商品.jpg から始まります。


## 「やり直し」コマンドとDriveの関係


- やり直し を入力すると、Driveの商品写真（02_以降）が全て削除されます
- テプラ画像（01_テプラ.jpg）は削除されません
- 削除後、新しい写真は02_商品.jpgから採番し直されます

# 8　困ったときは


### Q. テプラを貼り忘れた商品はどうすればよいか


- 管理番号をテキストで入力してセッションを開始してください
- テプラ写真がなくても商品写真の撮影・保存は行えます


### Q. 写真を撮り直したい


- 写真を投稿した後であればスレッドに やり直し と入力してください
- Driveの商品写真が全て削除され、1枚目から撮り直せます
- 完了後に撮り直したい場合は浅野に連絡してください


### Q. 完了後に写真を追加したい


- 完了後でも、同じスレッドに写真を投稿すれば自動的にDriveに追加されます
- Monday.comのステータスに影響はありません


### Q. 管理番号がわからない


- 分荷判定チャンネルの確定メッセージに管理番号が記載されています
- それでも不明な場合は先輩スタッフまたは浅野に確認してください


### Q. 「スレッドに返信」の方法がわからない


- スマートフォンの場合はメッセージを長押しすると「返信」が出ます
- PCの場合はメッセージにマウスを近づけると右側に「返信」アイコンが出ます


### Q. AIからの返答が来ない


- 1〜2分待ってから再度投稿してみてください
- 管理番号の形式が正しいか確認してください（例：2603-0001）
- 解決しない場合は浅野に連絡してください


## 問い合わせ先


| 困りごとの種類 | 連絡先 |
| --- | --- |
| システムの不具合・エラー | 浅野に連絡 |
| 管理番号がわからない | 先輩スタッフまたは浅野に確認 |
| Driveの写真を削除・修正したい | 浅野に依頼 |
| 撮影方法・品質の判断に迷う | @浅野儀頼 相談 で直接相談、または先輩スタッフに確認 |


わからないことは、何でも聞いてください。

  正確な写真の管理が、チーム全体の作業効率と販売品質を支えています。
