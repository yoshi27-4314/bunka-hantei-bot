# 1　新しい声とは


## この制度の目的


日々の仕事の中で気づいたこと・要望・アイデアを集めて、会社をもっと良くするための仕組みです。

声を出すとポイントがもらえます。ポイントは3ヶ月ごとにお給料に上乗せされます。


## ポイントの基本


- 1ポイント ＝ 100円
- 声を出すだけで 1ポイント もらえます
- 支払い：3月・6月・9月・12月のお給料と一緒

# 2　声の出し方


声の出し方は2つあります。どちらでもOKです。


## 方法① Slackで出す


ボットに直接メッセージを送ります。

書き方：

  新しい声 ○○○○○

「新しい声」の後にスペースを入れて、内容を書いてください。

入力例：
- 新しい声 倉庫の照明が暗いので明るくしてほしい
- 新しい声 梱包材の置き場所を変えたら作業しやすくなると思う
- 新しい声 出品の写真を撮るとき背景の布があるといいかも

送信すると、ボットから確認メッセージが届きます。


## 方法② フォームで出す


Slackを使っていない方はこちらの方法で出せます。

1. 以下のページを開く
   https://web-production-e7e9d.up.railway.app/voice

2. 自分の名前を選ぶ

3. 内容を入力して「送信」を押す

「送信しました」と表示されたら完了です。

# 3　ポイントの仕組み


声を出した後、浅野さんが内容を確認してポイントが加算されていきます。

| 段階 | 意味 | ポイント |
| --- | --- | --- |
| 投稿 | 声を出した時点で自動的にもらえる | +1pt（+100円） |
| 受理 | 浅野さんが「確かに」と受け取った | +1pt（+100円） |
| 採用 | 「やってみよう」と決まった | +1pt（+100円） |
| 実装 | 実際に会社のやり方が変わった | +3pt（+300円） |
| 優秀 | 大きな改善につながった声 | +15pt（+1,500円） |
| 見送り | 今回は見送り（また今度） | 0pt |
| 却下 | 対応が難しい | -1pt |


例：「倉庫の照明を明るくしてほしい」と声を出して、実際にLEDに交換された場合

  投稿（+1）→ 受理（+1）→ 採用（+1）→ 実装（+3）＝ 合計 6pt（600円）

# 4　声を出した後の流れ


1. あなたが声を出す
   → Slack or フォームで投稿。自動で +1pt

2. 浅野さんが確認する
   → 内容を見て「受理」「採用」などを判断

3. 進み具合がSlackで届く
   → ポイントが変わるたびにお知らせが届きます

4. 3ヶ月ごとにお給料に反映
   → 3月・6月・9月・12月のお給料と一緒にお支払い

# 5　どんな声を出せばいいか


何でもOKです。小さなことでも気づいたら出してください。

声は自動で「要望」「アイデア」「相談」のどれかに分類されるので、気にせず自由に書いてください。


## 要望（こうしてほしい）

- エアコンの温度をもう少し下げてほしい
- 梱包材の在庫が少なくなったら教えてほしい
- 作業手順書がほしい


## アイデア（こうしたらどうか）

- 商品写真を撮る場所を固定したら効率が上がりそう
- よく売れる商品をリスト化しておくと判定が早くなる
- 倉庫のレイアウトを変えたら動線がよくなる


## 相談（聞いてほしいこと）

- この作業のやり方が合っているか不安
- ○○の判断に迷うことが多い
- シフトについて相談したい

# 6　よくある質問


## 誰が声を出したか、みんなにわかりますか？

基本的にわかりません。毎朝の社内連絡では名前は出ません。ただし「実装」や「優秀」で大きく貢献した場合は、お名前付きで紹介されることがあります。


## 同じような内容を何度も出していいですか？

全く同じ内容はご遠慮ください。ただし「前に出した要望の補足」や「別の角度からの提案」は大歓迎です。


## 却下されてもポイントは減りますか？

投稿時の +1pt は残ります。却下で -1pt なので、差し引き 0pt になります。マイナスにはなりません。


## ポイントの確認方法は？

声を出すたびにSlackでお知らせが届きます。累計ポイントが気になる場合は浅野さんに聞いてください。
