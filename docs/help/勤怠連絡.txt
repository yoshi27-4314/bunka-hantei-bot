# 1　このチャンネルの役割


サイレント記録チャンネルは、遅刻・欠勤・早退などの勤怠連絡を記録するためのチャンネルです。

投稿した内容は自動でスプレッドシートに記録されます。


# 2　使い方


連絡内容をそのまま投稿してください。

例：
  明日遅刻します。10時出勤予定です。
  体調不良のため本日欠勤します。
  本日15時で早退します。


# 3　コマンド一覧


| コマンド | 動作 |
| --- | --- |
| （メッセージを投稿） | 勤怠連絡を記録 |
| ヘルプ | このメニューを表示 |
