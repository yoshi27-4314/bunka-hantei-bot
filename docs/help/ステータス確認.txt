# 1　このチャンネルの役割


ステータス松本チャンネルは、商品の現在の状態を確認するためのチャンネルです。

管理番号を送ると、Monday.comに登録されている情報を表示します。


# 2　使い方


2つの方法があります：

方法1：管理番号をテキストで送る
  2603-0001

方法2：管理番号が写った写真を送る（テプラの写真など）

旧形式の管理番号（2603V0001など）でも検索できます。


# 3　表示される情報


- 管理番号
- 現在のステータス（分荷確定 / 撮影完了 / 出品中 など）
- 判定チャンネル（eBayシングル / ヤフオクビンテージ など）
- 予想販売価格
- 在庫予測期間
- スコア
- 登録からの経過日数


# 4　コマンド一覧


| コマンド | 動作 |
| --- | --- |
| [管理番号] | ステータスを表示（例：2603-0001） |
| （写真を送る） | 写真から管理番号を読み取ってステータス表示 |
| ヘルプ | このメニューを表示 |
