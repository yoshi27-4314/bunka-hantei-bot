# 1　このチャンネルの役割


渋沢栄一チャンネルは、現場で出会った品物をその場で査定するためのチャンネルです。

3つの機能があります：
- 販売価格査定（写真を送るとAIが販売価格・スコアを算出）
- スクラップ計算（単価×重量の即時計算）
- 古物台帳（買取時の身分証確認・法定記録）

加えて、相場情報やメモの記録もできます。


# 2　販売価格査定


商品の写真やテキストを投稿すると、AIが以下を表示します：

- 販売・処分の選択肢（候補チャンネルの一覧）
- 予想販売価格
- 予想廃棄価格（処分にかかる費用）
- 総合スコア（100点満点・4項目内訳）
- 総評（箇条書き）

★買取価格はAIが決めません。販売価格とスコアを見て、人間が判断します。

使い方：
- 商品の写真を送る（最大3枚）
- またはテキストで商品情報を送る
- 写真＋テキストの組み合わせも可能


# 3　スクラップ計算


単価と重量を入力すると、合計金額を即座に計算します。

入力例：
  スクラップ 鉄 45円 120kg
  → 鉄 / ¥45/kg / 120kg / 合計 ¥5,400

素材名は省略できます：
  スクラップ 45円 80kg
  → 合計 ¥3,600


# 4　古物台帳


買取が決まったとき、古物営業法に基づく記録を作成します。

手順：
1. 「買取確定 ¥3000」と入力（金額は実際の買取価格）
2. 品物の名前・特徴を入力
3. 相手方の身分証の写真を送信
4. AIが自動で読み取り、内容を確認
5. 「登録」で古物台帳に記録

修正が必要な場合：
  修正 氏名：正しい名前
  修正 住所：正しい住所


# 5　現場メモ


「メモ」「相場」「業者」「単価」などのキーワードを含む投稿は、自動で記録されます。

例：
  メモ 鉄スクラップ 今月の単価キロ45円
  相場 銅線 キロ800円に値上がり
  業者 ○○リサイクル 連絡先090-xxxx-xxxx


# 6　コマンド一覧


| コマンド | 動作 |
| --- | --- |
| （写真を送る） | AIが販売価格を査定 |
| スクラップ [素材] [単価]円 [重量]kg | スクラップ金額を計算 |
| 買取確定 ¥[金額] | 古物台帳フローを開始 |
| 登録 | 古物台帳の確認内容を登録 |
| 修正 [項目]：[正しい値] | 古物台帳の内容を修正 |
| メモ [内容] | 現場メモを記録 |
| ヘルプ | このメニューを表示 |
| ヘルプ [番号] | 指定番号の内容を表示 |
| ヘルプ [キーワード] | キーワードで検索 |
