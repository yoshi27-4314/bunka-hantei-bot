# 1　このチャンネルの役割


二宮金次郎チャンネルは、出退勤を記録するためのチャンネルです。

出勤と退勤をこのチャンネルで申告すると、自動で勤務時間が計算されます。


# 2　出勤の仕方


チャンネルに「出勤」と投稿してください。

  出勤

出勤時刻が記録されます。


# 3　退勤の仕方


チャンネルに「退勤」と投稿してください。

  退勤

退勤時刻が記録され、勤務時間が自動計算されます。
休憩時間がある場合は自動で差し引かれます。


# 4　コマンド一覧


| コマンド | 動作 |
| --- | --- |
| 出勤 | 出勤を記録 |
| 退勤 | 退勤を記録（勤務時間を自動計算） |
| ヘルプ | このメニューを表示 |
