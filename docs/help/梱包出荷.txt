# 1　このチャンネルの役割


## 梱包出荷チャンネルとは


落札・入金確認が完了した商品を梱包→出荷手配→追跡番号登録するチャンネルです。


  管理番号を入力するだけで梱包に必要な情報（サイズ・販売チャンネル・棚番など）が表示され、
  追跡番号も送り状ラベルの写真1枚から自動登録されます。


## 絶対に守るルール


⚠️ 入金確認済みの商品のみ梱包すること


  入金確認前の商品を梱包してはいけません。

  Monday.com上のステータスが 「入金確認済み」 になってから、梱包作業を開始してください。


## 対象となる商品


✅ 新ボード品（通常）

    管理番号形式：`2603-0001`

    分荷判定システムで登録した商品。

    サイズ・チャンネル・予想価格が自動表示される。
  
  
    ✅ 旧ボード品（手作業登録）

    管理番号形式：`0612HK039`

    以前に手作業で登録した商品。

    アイテム名・棚番が表示される。


どちらの形式の管理番号も同じ手順で作業できます。

# 2　作業の流れ（5ステップ）


事前入金確認 → STEP 1管理番号入力 → STEP 2梱包完了 → STEP 3運送会社選択 → STEP 4ラベル撮影 → 完了出荷登録


- 梱包出荷チャンネルに管理番号を新規メッセージとして投稿する（またはテプラ写真を貼る）
- Botが梱包情報を返信したら、商品を梱包して 梱包完了 と入力する
- 運送会社を1〜5の番号で選ぶ
- 送り状ラベルの写真を撮って貼る（佐川・アート・西濃の場合）
- Botが追跡番号を読み取り、出荷登録が自動完了する


---

# 5　STEP 3：運送会社を選ぶ


## 5択メニューの内容と次のアクション


| 番号 | 運送会社 | 次のアクション |
| --- | --- | --- |
| 1️⃣ | 佐川急便 | 送り状ラベルの写真を送る → OCRで追跡番号を自動取得 |
| 2️⃣ | アートデリバリー | 同上 |
| 3️⃣ | 西濃運輸 | 同上 |
| 4️⃣ | 直接引き取り | 番号を選ぶだけで即完了。ラベル写真は不要 |

番号のみ入力してください。例：`1`

# 7　後日発送の場合


## 「5：後日発送」を選んだとき


今日は梱包だけ完了して、発送は後日行う場合に使用します。

  セッションはここで一度終了します。実際に発送したあと、以下の方法で伝票番号を登録してください。


## 後日の登録方法


梱包出荷チャンネルに以下の形式で新規メッセージを投稿してください：


```
管理番号　運送会社　伝票番号

例：
2603-0001 佐川 123456789012
0612HK039 アート 987654321
2602-0015 西濃 AB1234567890
```


| 項目 | 入力するキーワード |
| --- | --- |
| 佐川急便 | 佐川 |
| アートデリバリー | アート |
| 西濃運輸 | 西濃 |


⚠️ 管理番号・運送会社・伝票番号の順番を必ず守ってください。順番が違うと登録できません。

# 8　旧ボード品の出荷後について


## 旧ボード品はMonday.comが自動更新されません


旧ボード品（`0612HK039` 形式）は、Monday.comの列構造が新ボードと異なるため、
  出荷登録後にステータスが自動更新されません。


  代わりに、通販業務_共有コミュニティチャンネルに自動で通知が届きます。


## 自動通知の内容（共有コミュニティ宛）


Bot（通販業務_共有コミュニティ）
  @channel


  ━━━━━━━━━━━━━━━━

  📦 旧ボード品が出荷されました

  ━━━━━━━━━━━━━━━━

  🔖 管理番号：0612HK039

  📋 アイテム名：ヴィンテージ茶碗 北大路魯山人作

  🏢 運送会社：佐川急便

  📮 追跡番号：123456789012


  ━━━━━━━━━━━━━━━━

  ✅ 対応をお願いします

# 9　コマンド一覧


| コマンド | タイミング | 動作 |
| --- | --- | --- |
| 梱包完了 / 梱包 | 梱包が終わったとき | 運送会社選択メニューを表示する |
| 1 〜 5 | 運送会社選択時 | 選択した運送会社を登録し、次のステップへ進む |
| 削除 / キャンセル | 作業を取り消したいとき | 現在のセッションをキャンセルする。記録はされない |


コマンドはすべて、Botが返信したスレッド内で入力してください。新規メッセージでは動作しません。

# 11　困ったときは


## よくある質問


| 状況 | 対処方法 |
| --- | --- |
| 管理番号を入力してもBotが反応しない | 新規メッセージとして投稿しているか確認してください。スレッド返信では開始されません。 |
| 追跡番号の読み取りに連続して失敗する | ラベルに近づき、数字部分のみを正面から撮影してください。それでも失敗する場合は浅野に連絡してください。 |
| 旧ボード品の管理番号が見つからないと言われる | 管理番号の文字が正確か確認してください。テプラ写真で送ると認識精度が上がります。 |
| 誤った管理番号でセッションを始めてしまった | スレッド内で キャンセル または 削除 と入力してセッションを終了し、正しい管理番号で新規投稿し直してください。 |
| 後日発送で伝票番号を入力したがBotが反応しない | 「管理番号 運送会社 伝票番号」の順番と半角スペース区切りを確認してください。 |
